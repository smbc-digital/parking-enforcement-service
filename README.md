## Requirements
- dotnet core 3.1
- editor of your choice
- gpg key added to accepted contributors


## Installation
```console
$ git clone git@github.com:smbc-digital/parking-enforcement-service.git
$ cd parking-enforcement-service/src
$ git-crypt unlock
$ dotnet run
```
